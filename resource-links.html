Curated Resources to Boost Your Learning

We've hand-picked a selection of external websites, documentation, and tools to help you deepen your understanding and expand your coding skills beyond our core lessons.

HTML & Web Structure

  • MDN Web Docs - HTML: Comprehensive HTML reference.
  • HTML.com: Beginner-friendly guides and tutorials.
  • W3Schools HTML Tutorial: Interactive examples and exercises.
  • W3C Markup Validation Service: Check your HTML for errors.

CSS & Web Styling

  • MDN Web Docs - CSS: In-depth CSS documentation.
  • CSS-Tricks: Articles, guides, and snippets for all things CSS.
  • W3Schools CSS Tutorial: Practical CSS examples and references.
  • Flexbox Froggy: A game to learn CSS Flexbox.
  • CSS Grid Garden: A game to learn CSS Grid.

JavaScript & Interactivity

  • MDN Web Docs - JavaScript: Core JavaScript language reference.
  • The Modern JavaScript Tutorial: A thorough, modern JS guide.
  • W3Schools JavaScript Tutorial: Interactive examples.
  • Eloquent JavaScript: A free online book covering JS deeply.

Python & Data Science

  • Python Official Documentation: Comprehensive reference.
  • W3Schools Python Tutorial: Easy-to-understand basics.
  • Real Python: In-depth tutorials and articles.
  • Automate the Boring Stuff with Python: Practical applications for beginners.

Java & Object-Oriented Programming

  • Oracle Java Documentation: Official Java SE documentation.
  • W3Schools Java Tutorial: Quick start and examples.
  • TutorialsPoint Java: Broad range of Java topics.
  • JavaTpoint: Detailed Java tutorials with examples.

General Tools & Concepts

  • Stack Overflow: The go-to Q&A site for developers.
  • GitHub: Learn version control and collaborate on projects.
  • freeCodeCamp: Interactive coding lessons and certifications.
  • Codewars: Improve your coding skills through challenges.
  • VS Code Online: A lightweight online code editor.
